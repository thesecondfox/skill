# Chemoinformatics & Drug Discovery

## 分析流程

### 1. 分子操作
- **bio-chemoinformatics-molecular-io**: 分子文件读写（SDF、MOL2）
- **bio-chemoinformatics-molecular-descriptors**: 分子描述符计算

### 2. 虚拟筛选
- **bio-chemoinformatics-similarity-searching**: 分子相似性搜索
- **bio-chemoinformatics-virtual-screening**: 虚拟筛选流程

### 3. ADMET 预测
- **bio-chemoinformatics-admet-prediction**: 药物性质预测

### 4. 工具库
- **rdkit**: 化学信息学核心库
- **deepchem**: 深度学习药物发现
- **datamol**: 分子数据处理
