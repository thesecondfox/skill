# Machine Learning for Omics

## 分析流程

### 1. 分类与预测

### 2. 模型验证

### 3. 工具库
- **scikit-learn**: 机器学习核心库
- **pytorch-lightning**: PyTorch 训练框架
- **transformers**: 预训练模型
