# Genomics Pipeline

## 分析流程

### 1. 变异检测
- **bio-variant-calling-gatk-variant-calling**: GATK 最佳实践流程
- **bio-variant-calling-deepvariant**: DeepVariant 深度学习变异检测
- **bio-variant-calling-variant-calling**: 通用变异检测流程
- **bio-variant-calling-joint-calling**: 联合变异检测

### 2. 变异注释与过滤
- **bio-variant-calling-variant-annotation**: VEP、SnpEff 功能注释
- **bio-variant-calling-filtering-best-practices**: GATK 过滤最佳实践
- **bio-variant-calling-variant-normalization**: 变异标准化

### 3. 结构变异
- **bio-variant-calling-structural-variant-calling**: SV 检测（Manta、LUMPY）
- **bio-copy-number-cnvkit-analysis**: CNV 检测
- **bio-copy-number-gatk-cnv**: GATK CNV 分析

### 4. 群体遗传学
- **bio-population-genetics-plink-basics**: PLINK 基础操作
- **bio-population-genetics-population-structure**: PCA、ADMIXTURE 群体结构
- **bio-population-genetics-association-testing**: GWAS 关联分析
- **bio-population-genetics-linkage-disequilibrium**: LD 分析

### 5. 基因分型与填充
- **bio-phasing-imputation-haplotype-phasing**: 单倍型定相
- **bio-phasing-imputation-genotype-imputation**: 基因型填充

### 6. 基因组组装
- **bio-genome-assembly-short-read-assembly**: 短读长组装
- **bio-genome-assembly-long-read-assembly**: 长读长组装
- **bio-genome-assembly-assembly-qc**: 组装质量评估

### 7. 基因组注释
- **bio-genome-annotation-prokaryotic-annotation**: 原核生物注释（Prokka）
- **bio-genome-annotation-eukaryotic-gene-prediction**: 真核生物基因预测
- **bio-genome-annotation-functional-annotation**: 功能注释
