# Immunology & Immune Repertoire Pipeline

## 分析流程

### 1. TCR/BCR 分析
- **bio-tcr-bcr-analysis-mixcr-analysis**: MiXCR TCR/BCR 重建
- **bio-tcr-bcr-analysis-immcantation-analysis**: Immcantation 工具链
- **bio-tcr-bcr-analysis-scirpy-analysis**: 单细胞免疫组库分析

### 2. 流式细胞术
- **bio-flow-cytometry-fcs-handling**: FCS 文件读取
- **bio-flow-cytometry-gating-analysis**: 门控分析
- **bio-flow-cytometry-clustering-phenotyping**: 聚类与表型分析

### 3. 免疫信息学
- **bio-immunoinformatics-epitope-prediction**: 表位预测
- **bio-immunoinformatics-mhc-binding-prediction**: MHC 结合预测
- **bio-immunoinformatics-neoantigen-prediction**: 新抗原预测
